simplesvgmap by zmmbreeze

This project can be used to build map application on web by svg and js!

This project help programmer easily build map application on web. This project is base on MIT or GPL license. Things can do: build map on web.(support Firefox 3.0 and upper,chrome 5 and upper, adobe svg on IE); moveMap; zoomIn; zoomOut; set the width and height of map; add object(image,text,circle) to map; remove object from map; rasterize(generate the string of svg map);
